Arkusze CSS

Arkusz lokalny

Arkusz znajdujący się przy znaczniku który chcemy sformaować.

    
        <h1 style="color:red; font-size:12px; margin:5px;"> ... <⁄h1>

Arkusz wewnętrzny

Arkusz znajdujący się w dokumencie HTML w sekcji <head><⁄head>

    
        <head>
        <style>
            body {
                background-color: blue;
            }
            h1 {
                color: red;
                margin-left: 40px;
            }
        <⁄style>
        <⁄head>

Arkusz zewnętrzny

Arkusz znajdujący się w oddzielnym pliku. W dokumencie HTML umieszczany jest jedynie link do niego.

    
        body {
            background-color: lightblue;
        }
        h1 {
            color: navy;
            margin-left: 20px;
        }

Odnośnik do pliku "styles.css"

    
        <head>
            <link rel="stylesheet" type="text/css" href="styles.css">
        <⁄head>
Kaskadowość
Im styl jest bliżej miejsca, którego dotyczy, tym jest ważniejszy.
Reguła
Instrukcja formatująca, mająca zastosowanie na stronie WWW.
    
        body {
            background-color: lightblue;
        }
